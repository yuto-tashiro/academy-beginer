# フリーランスセラピストアカデミー・ビギナー研修

フリーランスセラピストとして一歩目を進めるための自動化学習システム

## 🎯 概要

5本の動画コンテンツとそれぞれに対応するアンケートを順番に進めていく学習プラットフォームです。進捗は自動的に保存され、いつでも中断・再開できます。

## ✨ 主な機能

- **📹 動画視聴**: YouTube動画の埋め込み表示（全5レッスン）
- **📝 アンケート**: 各レッスン後の詳細なアンケートフォーム
- **💾 自動保存**: localStorageによる進捗の自動保存
- **🎨 モダンなUI**: グラスモーフィズムとダークテーマ
- **📱 レスポンシブ**: モバイル・タブレット・デスクトップ対応
- **⌨️ キーボード対応**: Alt + ← / → でナビゲーション

## 🚀 使い方

### ローカルで実行

```bash
# プロジェクトディレクトリに移動
cd "フリーランスセラピストアカデミー（ビギナー）"

# ローカルサーバーを起動
python3 -m http.server 8000
```

ブラウザで http://localhost:8000 にアクセス

### GitHub Pagesでデプロイ

1. このリポジトリをフォークまたはクローン
2. GitHub リポジトリの Settings → Pages
3. Source を `main` ブランチに設定
4. 公開URLにアクセス

## 📚 レッスン内容

### レッスン1: アイデアの発見と可能性の探求
サービスアイデアの発見と、クライアントのニーズを理解する

### レッスン2: ビジネスモデルの具体化
価格設定、活動エリア、営業方法などの具体化

### レッスン3-5: 学びと気づき
各動画から得られる学びと気づきを深める

## 🛠️ 技術スタック

- **HTML5**: セマンティックな構造
